## [1.0.0] - 2024-05-14

### Added
- Initial release of create-skill meta-skill.
- Dual-path workflow: manual (agent tools only) + optional Python accelerators.
- Frontmatter spec with version, author, tags, requires/related dependencies.
- `suggests` field for optional dependencies (tools, runtimes, mcps).
- Five accelerator scripts: init_skill, validate_skill, run_eval, improve_description, install_to_platform.
- Four reference documents: frontmatter-spec, platform-differences, description-patterns, eval-methodology.
- Templates for SKILL.md and evals.json.
- Public skill template (skill-template.md) with canonical structure.
- Pre-publish checklist asset.
- Self-validation: validator passes on create-skill itself.
- **Pre-execution Check** and **Safety Boundaries** (Forbidden Operations,
  Confirmation Gates, Emergency Stop) are mandatory for all generated skills.
- **ALL output MUST be in English.** Hard Rule enforced for all deliverables.
